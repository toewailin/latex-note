# LaTex Command Syntax

LaTeX မှာ command တွေကို ပုံသေ syntax တစ်ခုအနေနဲ့ သတ်မှတ်ထားပါတယ်။ အခြေခံအားဖြင့် command တစ်ခုဟာ backslash `\` နဲ့ စတင်ရပါတယ်။

LaTeX command တွေကို နားလည်လွယ်အောင် အဓိက အစိတ်အပိုင်း (၃) ပိုင်းနဲ့ ခွဲခြားရှင်းပြပေးပါမယ်။

---

## ၁။ Command တည်ဆောက်ပုံ (Syntax)
အခြေခံပုံစံကတော့ ဒီလိုပါ-
> `\commandname[optional]{required}`

* **`\` (Backslash):** ဒါက LaTeX ကို "ဒါဟာ စာသားမဟုတ်ဘူး၊ ခိုင်းစေချက် (Command) ဖြစ်တယ်" လို့ ပြောတာပါ။
* **`commandname`:** ခိုင်းစေချင်တဲ့ အလုပ်အမည် (ဥပမာ- `section`, `textbf`)။
* **`[]` (Square Brackets):** မဖြစ်မနေ ထည့်စရာမလိုတဲ့ Option တွေအတွက် သုံးပါတယ်။
* **`{}` (Curly Brackets):** မဖြစ်မနေ ထည့်ပေးရမယ့် အချက်အလက် (Argument) အတွက် သုံးပါတယ်။

---

## ၂။ အသုံးအများဆုံး အခြေခံ Commands များ

### (က) စာသား ပုံစံချခြင်း (Text Formatting)
* `\textbf{စာသား}`: စာသားကို **အထူ (Bold)** လုပ်ပေးပါတယ်။
* `\textit{စာသား}`: စာသားကို *အစောင်း (Italic)* လုပ်ပေးပါတယ်။
* `\underline{စာသား}`: စာသားအောက်မှာ မျဉ်းတားပေးပါတယ်။
* `\huge{စာသား}`, `\large{စာသား}`, `\small{စာသား}`: စာလုံးအရွယ်အစား ပြောင်းလဲပေးပါတယ်။

### (ခ) စာမျက်နှာ တည်ဆောက်ပုံ (Structure)
* `\section{ခေါင်းစဉ်}`: အခန်းခေါင်းစဉ် အကြီး (ဥပမာ- 1. Introduction)။
* `\subsection{ခေါင်းစဉ်}`: အခန်းခွဲ ခေါင်းစဉ် (ဥပမာ- 1.1 Background)။
* `\newpage`: စာမျက်နှာအသစ် တစ်ခုကို ကူးပြောင်းစေပါတယ်။
* `\tableofcontents`: စာအုပ်ရဲ့ မာတိကာကို အလိုအလျောက် ထုတ်ပေးပါတယ်။

---

## ၃။ Environments (စတင်ခြင်းနှင့် ပိတ်ခြင်း)
Command အချို့ဟာ စာကြောင်းတစ်ကြောင်းတည်းတင် မကဘဲ Block တစ်ခုလုံးကို အကျိုးသက်ရောက်စေချင်တဲ့အခါ `\begin` နဲ့ `\end` ကို သုံးရပါတယ်။ ဒါကို **Environment** လို့ ခေါ်ပါတယ်။

ဥပမာ- စာရင်းပြုစုခြင်း (Bullet Points):
```latex
\begin{itemize}
    \item ပထမအချက်
    \item ဒုတိယအချက်
\end{itemize}
```

---

## ၄။ သင်္ချာဆိုင်ရာ Commands (Mathematical)
LaTeX ရဲ့ အကောင်းဆုံး အစိတ်အပိုင်းပါ။ သင်္ချာ command တွေကို `$ $` ကြားထဲမှာ ရေးရပါတယ်။

* **အမြှောက်/အစား:** `$a \times b$`, `$a \div b$`
* **အပိုင်းကိန်း:** `\frac{အပေါ်ကိန်း}{အောက်ကိန်း}` ဥပမာ- `\frac{1}{2}` ($\frac{1}{2}$)
* **အညွှန်းကိန်း (Power):** `x^{2}` ($x^2$)
* **Root:** `\sqrt{x}` ($\sqrt{x}$)
* **Greek Letters:** `\alpha`, `\beta`, `\gamma`, `\pi` ($\alpha, \beta, \gamma, \pi$)

## ၅။ သိထားသင့်တဲ့ အထူးသင်္ကေတများ
အချို့ သင်္ကေတတွေက LaTeX ရဲ့ command တွေမှာ ပါဝင်နေတဲ့အတွက် စာသားအဖြစ် ရိုက်ချင်ရင် `\` ခံရပါတယ်။
* `\%` လို့ ရိုက်မှ **%** ပေါ်မှာပါ။
* `\$` လို့ ရိုက်မှ **$** ပေါ်မှာပါ။
* `\&` လို့ ရိုက်မှ **&** ပေါ်မှာပါ။

### အကြံပြုချက်
Overleaf မှာ စမ်းကြည့်တဲ့အခါ `\section{...}` တို့ `\textbf{...}` တို့ကို အရင်ဆုံး စမ်းရိုက်ကြည့်ပါ။ အမှားတစ်ခုခုရှိရင် Overleaf က ဘယ်နားမှာ မှားနေလဲဆိုတာကို **Error message** ပြပေးပါလိမ့်မယ်။

## ၆။ ပုံများ ထည့်သွင်းခြင်း (Inserting Images)
ပုံတွေထည့်ဖို့အတွက် `graphicx` package ကို preamble ထဲမှာ အရင်ဆုံး ထည့်ထားရပါမယ်။

### အခြေခံ Command:
```latex
\begin{figure}[h] % [h] ဆိုတာ 'here' (ဒီနေရာမှာပဲ ထားပါ) လို့ ပြောတာပါ
    \centering % ပုံကို အလယ်ပို့မယ်
    \includegraphics[width=0.7\textwidth]{filename} % ပုံရဲ့ အကျယ်နဲ့ ဖိုင်နာမည်
    \caption{ပုံရဲ့ ခေါင်းစဉ်} % ပုံအောက်မှာ ပေါ်မယ့်စာသား
    \label{fig:my_image} % နောက်ပိုင်းမှာ ပြန်ညွှန်းဖို့ ID ပေးတာ
\end{figure}
```

* **`[h]` Option:** ပုံကို ကိုယ်ရေးတဲ့နေရာမှာပဲ ကပ်နေစေချင်ရင် သုံးပါတယ်။ တခြား options တွေက `[t]` (top) နဲ့ `[b]` (bottom) တို့ ဖြစ်ပါတယ်။
* **`width=...`:** စာမျက်နှာအကျယ်ရဲ့ ဘယ်နှဆလဲဆိုတာ သတ်မှတ်တာပါ။ (ဥပမာ- `0.5\textwidth` ဆိုရင် စာမျက်နှာရဲ့ တစ်ဝက်ပဲ ကျယ်ပါမယ်)။

## ၇။ ဇယားများ ဆွဲခြင်း (Creating Tables)
LaTeX မှာ ဇယားဆွဲတာက Word ထက်စာရင် နည်းနည်း ပိုရှုပ်ပါတယ်။ ဒါပေမဲ့ ပိုပြီး တိကျပါတယ်။

### အခြေခံ ဇယားပုံစံ:
```latex
\begin{table}[h]
    \centering
    \caption{နမူနာ ဇယား}
    \begin{tabular}{|c|l|r|} % ကော်လံ ၃ ခု (center, left, right align)
        \hline % အပေါ်ဆုံးက မျဉ်းတား
        စဉ် & အမည် & ရမှတ် \\ \hline \hline % ခေါင်းစဉ်တန်း
        ၁ & မောင်မောင် & ၈၅ \\ \hline
        ၂ & အောင်အောင် & ၉၀ \\ \hline
    \end{tabular}
\end{table}
```

* **`{|c|l|r|}`:**
    * `|` ဆိုတာ ဒေါင်လိုက်မျဉ်းတား (Vertical Line) ပါ။
    * `c` (center), `l` (left), `r` (right) align လုပ်တာပါ။
* **`&` (Ampersand):** ကော်လံတစ်ခုနဲ့ တစ်ခုကြား ခွဲတဲ့ သင်္ကေတပါ။
* **`\\` (Double Backslash):** နောက်တစ်ကြောင်းကို ဆင်းတာပါ။
* **`\hline`:** အလျားလိုက် မျဉ်းတား (Horizontal Line) ဆွဲတာပါ။

## ၉။ Professional Tips
* **Cross-Referencing:** စာထဲမှာ "ပုံ (၁) ကို ကြည့်ပါ" လို့ ရေးချင်ရင် `As shown in Figure \ref{fig:my_image}` လို့ ရေးရပါတယ်။ ဒါဆိုရင် LaTeX က ပုံနံပါတ်ကို အလိုအလျောက် တပ်ပေးပါလိမ့်မယ်။
* **Subfigures:** ပုံတစ်ပုံတည်းမှာပဲ ဘေးချင်းယှဉ်ပြီး ပုံအသေးလေးတွေ ပြချင်ရင် `subcaption` package ကို သုံးရပါတယ်။
