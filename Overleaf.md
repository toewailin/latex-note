Overleaf က LaTeX ကို Software တွေ Install လုပ်စရာမလိုဘဲ Browser ကနေတင် အလွယ်တကူ သုံးနိုင်တဲ့အတွက် စတင်လေ့လာသူတွေအတွက် အကောင်းဆုံးပါ။

Overleaf ကို စနစ်တကျ အသုံးချတတ်ဖို့ အခြေခံအဆင့် (၅) ဆင့်နဲ့ ရှင်းပြပေးပါမယ်။

---

## ၁။ အကောင့်ဖွင့်ခြင်းနှင့် Project စတင်ခြင်း
1.  [Overleaf.com](https://www.overleaf.com) ကိုသွားပြီး Google Account (သို့မဟုတ်) Email နဲ့ **Register** လုပ်ပါ။
2.  Dashboard ရောက်ရင် **"New Project"** ခလုတ်ကို နှိပ်ပါ။
3.  **"Blank Project"** ကို ရွေးပြီး နာမည်တစ်ခု ပေးလိုက်ပါ (ဥပမာ- `MyFirstProject`)။

---

## ၂။ Overleaf ရဲ့ Interface ကို နားလည်ခြင်း
မျက်နှာပြင်မှာ အဓိက အပိုင်း (၃) ပိုင်း ပါရှိပါတယ် -
* **ဘယ်ဘက်ခြမ်း (File Tree):** ကိုယ့် Project ထဲမှာရှိတဲ့ `.tex` ဖိုင်တွေ၊ ပုံတွေ (Images) နဲ့ Bibliography ဖိုင်တွေကို စီမံတဲ့နေရာပါ။
* **အလယ် (Editor):** LaTeX code တွေ ရေးရမယ့်နေရာပါ။
* **ညာဘက်ခြမ်း (Preview):** **"Recompile"** နှိပ်လိုက်ရင် ထွက်လာမယ့် PDF output ကို ကြည့်ရမယ့်နေရာပါ။

---

## ၃။ အခြေခံ Code တည်ဆောက်ပုံ (Structure)
Project တစ်ခုစရင် အနည်းဆုံး ဒီ Code တွေ ပါရပါမယ်။ ဒါကို Editor ထဲမှာ ရိုက်ထည့်ကြည့်ပါ -

```latex
\documentclass{article} % စာရွက်စာတမ်း အမျိုးအစား

% --- Preamble (Setting များ ထည့်သွင်းသည့်နေရာ) ---
\usepackage[utf8]{inputenc} % Encoding သတ်မှတ်ချက်
\title{ကျွန်တော့်ရဲ့ ပထမဆုံး LaTeX စာတမ်း}
\author{Toe Wai Lin}
\date{April 2026}

% --- Document Body (စာသားများ ရေးသားသည့်နေရာ) ---
\begin{document}

\maketitle % ခေါင်းစဉ်ကို စာမျက်နှာပေါ်မှာ ဖော်ပြခိုင်းခြင်း

\section{နိဒါန်း}
ဒါကတော့ LaTeX ကို Overleaf သုံးပြီး စမ်းသပ်ရေးသားခြင်း ဖြစ်ပါတယ်။ 

\section{သင်္ချာ ညီမျှခြင်းများ}
LaTeX ရဲ့ အားသာချက်က သင်္ချာပုံသေနည်းတွေကို ရိုက်ရတာ လွယ်ကူတာပါ။
ဥပမာ- $a^2 + b^2 = c^2$

\end{document}
```

## ၄။ ပုံများနှင့် Table များ ထည့်သွင်းခြင်း
Overleaf မှာ ပုံတစ်ပုံ ထည့်ချင်ရင် -
1.  ဘယ်ဘက် File Tree က **"Upload"** icon လေးကို နှိပ်ပြီး ကိုယ့်စက်ထဲက ပုံကို အရင်တင်ပါ။
2.  Code ထဲမှာ `\usepackage{graphicx}` ကို အပေါ်ဆုံးမှာ ထည့်ပါ။
3.  ပုံပြလိုတဲ့ နေရာမှာ အောက်ပါအတိုင်း ရေးပါ -
```latex
\begin{figure}[h]
    \centering
    \includegraphics[width=0.5\textwidth]{image_name} % ပုံနာမည်ထည့်ပါ
    \caption{ဒါကတော့ နမူနာပုံဖြစ်ပါတယ်။}
\end{figure}
```

## ၅။ Overleaf ရဲ့ အသုံးဝင်သော Shortcut များနှင့် Tips
* **Ctrl + Enter (သို့မဟုတ် Cmd + Enter):** ဒါက **Recompile** လုပ်တဲ့ Shortcut ပါ။ Code ပြင်ပြီးတိုင်း ဒါကို နှိပ်ပြီး Output ကြည့်ပါ။
* **Auto-complete:** LaTeX command တွေ ရိုက်တဲ့အခါ `\` လေးစရိုက်တာနဲ့ သူက command တွေကို အလိုအလျောက် ညွှန်းပေးပါလိမ့်မယ်။
* **History:** ဘယ်ဘက်အပေါ်မှာ "History" ဆိုတာ ရှိပါတယ်။ အရင်က ရေးခဲ့တာတွေ မှားသွားလို့ ပြန်ပြင်ချင်ရင် အချိန်မရွေး နောက်ပြန်ဆုတ် (Restore) လုပ်လို့ ရပါတယ်။
* **Share:** ညာဘက်အပေါ်က "Share" ခလုတ်ကို နှိပ်ပြီး တခြားသူတွေကို link ပေးပြီး တူတူရေးခိုင်းလို့ ရပါတယ်။

### သင်စမ်းကြည့်ဖို့ အကြံပြုချက်
အခု Overleaf မှာ Project အသစ်တစ်ခုဖွင့်ပြီး အပေါ်ကပေးထားတဲ့ Code လေးကို ကူးထည့်ကြည့်ပါ။ ပြီးရင် **"Recompile"** နှိပ်ကြည့်ပါ။ PDF ထွက်လာပြီဆိုရင် သင် LaTeX ကို စတင်တတ်မြောက်နေပြီ ဖြစ်ပါတယ်။
